Install LUMEN on your Chromebook

Same voice features as Windows — Tesco, search, scroll — using Google Chrome (not Edge).

  1. Unzip LUMEN-Chromebook if you haven’t already. Keep the lumen-extension folder.
  2. Open Chrome. In the address bar type: chrome://extensions and press Enter.
  3. Turn Developer mode ON (toggle top-right). This is free — it only lets you install LUMEN from your files. You are not publishing anything.
  4. Click Load unpacked.
  5. Select the lumen-extension folder (the one with manifest.json inside).
  6. Click the puzzle icon → Pin “LUMEN Mind”.
  7. Click the LUMEN icon → side panel opens → tap Tap to speak.
  8. Allow the microphone when Chrome asks.

Try saying

  • open Tesco
  • search spaghetti
  • scroll then stop then up

Why not the Chrome Web Store? Google charges developers a one-time fee to publish. LUMEN is distributed free as a ZIP — you install it yourself in one minute. No subscription, no store, no cost.
